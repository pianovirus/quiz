# 기출문제 데이터 폴더

이 폴더에 **연도별 파일 하나씩**을 두면 앱이 자동으로 모두 로드합니다.
별도 등록·import 작업이 필요 없습니다.

## 새 연도 추가하는 방법

1. 이 폴더에 `YYYY.js` 형식으로 파일 생성 (예: `2024.js`, `2023.js`)
2. 아래 템플릿에 맞춰 문제를 작성
3. 저장하면 Vite HMR이 자동 반영

## 파일 템플릿

```js
// 2024년 기출문제
export default [
  {
    id: 1,                          // 같은 연도 내에서만 고유하면 됨
    subject: "감리 및 사업관리",     // 아래 5개 중 하나
    question: "문제 본문...",
    options: [
      "보기 1",
      "보기 2",
      "보기 3",
      "보기 4"
    ],
    answer: 4                        // 정답 보기 번호 (1~4)
  },
  // ...
];
```

## subject 값 (정확히 일치해야 함)

- `"감리 및 사업관리"`
- `"소프트웨어 공학"`
- `"데이터베이스"`
- `"시스템 구조"`
- `"보안"`

## id 처리

각 파일 안에서만 고유하면 됩니다. 로더가 자동으로
`"2025-1"`, `"2024-1"` 처럼 연도 prefix를 붙여 전역 고유 id를 만듭니다.

## 주의사항

- 파일명은 반드시 4자리 숫자 + `.js` (정규식 `[0-9]{4}\.js`)
- 작은따옴표 안에 작은따옴표가 들어가면 큰따옴표로 감싸거나 `\'`로 escape
- 한글 따옴표(`"`, `"`)는 문자열 종료가 아니므로 그대로 사용 가능
