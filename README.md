# 기출문제 랜덤 풀이 앱

연도별 기출문제를 랜덤으로 풀어볼 수 있는 학습용 웹앱입니다.

## 🚀 실행 방법

### 1. Node.js 설치 (이미 있으면 건너뛰기)

[https://nodejs.org](https://nodejs.org) 에서 LTS 버전을 다운로드해서 설치하세요. (18 이상 권장)

설치 확인:
```bash
node -v
```

### 2. VS Code에서 폴더 열기

이 폴더(`quiz-project`)를 VS Code로 열어주세요.

- VS Code 실행 → `파일` → `폴더 열기` → 이 폴더 선택
- 또는 폴더 우클릭 → `Code(으)로 열기`

### 3. 터미널에서 명령어 실행

VS Code에서 단축키 `` Ctrl + ` `` (백틱) 으로 터미널을 열고 아래 명령어를 차례로 입력하세요.

```bash
# 1) 필요한 패키지 설치 (처음 한 번만)
npm install

# 2) 개발 서버 실행
npm run dev
```

설치는 1~2분 정도 걸려요. 완료되면 자동으로 브라우저가 열리고 앱이 실행됩니다!

브라우저가 자동으로 안 열리면 터미널에 표시되는 주소(보통 `http://localhost:5173`)를 직접 들어가시면 돼요.

### 4. 종료

터미널에서 `Ctrl + C` 누르면 서버가 종료됩니다.

## 📚 앱 기능

- **랜덤 출제**: 선택한 과목에서 한 문제씩 랜덤 출제
- **과목별 필터**: 5개 과목(감리/SW공학/DB/시스템/보안) 중 원하는 것만 선택
- **즉시 채점**: 보기 클릭 시 바로 정답/오답 표시
- **틀린 문제 다시 풀기**: 틀린 문제는 자동 저장되어 별도 모드로 풀이 가능
- **북마크**: 다시 보고 싶은 문제 표시
- **학습 통계**: 전체 정답률 + 과목별 정답률 표시
- **영구 저장**: 학습 기록은 브라우저에 자동 저장 (새로고침해도 유지)

## 📁 프로젝트 구조

```
quiz-project/
├── index.html              # HTML 진입점
├── package.json            # 의존성 정보
├── vite.config.js          # Vite 설정
├── src/
│   ├── main.jsx           # React 진입점
│   └── App.jsx            # 메인 앱 (모든 문제 데이터 + UI)
└── README.md
```

문제 데이터를 수정하려면 `src/App.jsx` 상단의 `QUESTIONS` 배열을 편집하세요.

## ⚠️ 알아두실 점

- 일부 문제(22번, 26번, 38번, 50번, 79번 등)는 원본에 그림/표/다이어그램이 포함되어 있어 텍스트로 풀어 적었습니다. 필요시 PDF를 함께 참고하세요.
